Intent-Based Control — Point, Confirm, Execute

Intent-Based Control for
Golf Course Mobility

POINGO is a patent-pending, intent-based control system developed by California Suncap LLC. It changes how golfers interact with mobility devices — from button-driven operation to expressing intent through pointing and confirmation.

Golfers point to express direction or behavior intent, then confirm to authorize execution. The system handles motion details — stability, speed control, obstacle awareness, and terrain-appropriate behavior — on real golf courses with slopes, paths, and uneven ground.

What We’re Building

POINGO is building a unified, intent-based control platform designed specifically for golf-course environments — from walking push carts to low-speed golf carts.

  • Smart Follow Push Cart — follow that adapts to terrain, obstacles, and course behavior.
  • Terrain-Aware Golf Cart — assisted manual control with stable, predictable behavior on slopes.

The same core system interprets user intent through pointing and confirmation, then translates that intent into smooth, constrained, and context-aware vehicle behavior.

  • Point to express intent, confirm to authorize action
  • Assisted manual control (less micro-steering)
  • Decision-aware follow (calmer behavior in complex areas)
  • Built for real courses: slopes, paths, crossings
